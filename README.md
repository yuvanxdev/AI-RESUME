# AI Resume Analyzer / ATS Tracker

An AI-powered resume analysis system that evaluates resumes against job descriptions using ATS (Applicant Tracking System) scoring. It helps users understand how well their resume matches a job and provides improvement suggestions.

## Features

- ATS score calculation
- Resume and job description matching
- Keyword and skill extraction
- Resume parsing (PDF/DOCX support)
- Improvement suggestions based on analysis
- Dashboard for results visualization

## Tech Stack

- Frontend: HTML, CSS, JavaScript / React
- Backend: Node.js / Java / Spring Boot
- AI/NLP: Natural Language Processing / OpenAI APIs (if used)
- Database: MongoDB / MySQL (if used)
